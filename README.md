# 🚀 Instagram Unfollower Tracker

A fast, privacy-first tool to see who unfollowed you on Instagram — instantly, with no login required.

👉 Try it here: https://unfollowertracking.com

---

## 🔍 What is this?

This project powers a **simple unfollower tracking system** that helps you understand who unfollowed you on Instagram without needing to log in or install any app.

Most existing tools require:
- Instagram login
- permissions access
- or mobile app installation

This tool removes all of that friction.

---

## ✨ Key Features

- ⚡ Instant unfollower detection
- 🔒 No login required
- 🧠 Works with Instagram export data
- 📱 No app installation needed
- 🌐 Fully browser-based
- 🆓 Free to use

---

## 🧠 How it works

1. Upload or process Instagram export data  
2. Extract follower & following lists  
3. Compare both datasets  
4. Show users who don’t follow you back or unfollowed you
